## Spliting Component By Feature & State
**We still use on project of App.jsx file**

Before we have CoreConcept.jsx file, and now we add or create file **CoreConcepts.jsx** (using s ) and also create file of **Examples.jsx** in folder **components**

And in file of *CoreConcepts.jsx* we can paste the code from *App.jsx* and import some data :

```javascript
import CoreConcept from "./CoreConcept.jsx";
import {CORE_CONCEPTS} from '../data.js';

export default function CoreConcepts() {
    return (
        <section id="core-concepts">
          <ul>
            {CORE_CONCEPTS.map((conceptItem) =>(
              <CoreConcept key={conceptItem.title} {...conceptItem} />
            ))}
          </ul>
        </section>
    );
}
```
